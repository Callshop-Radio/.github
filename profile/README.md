# Callshop Radio

Community internet radio from Düsseldorf, Leipzig and Vienna — live Mon–Sun.

🎧 **Listen at [callshopradio.com](https://callshopradio.com)**

We're publishing the code that runs the station as inspiration for other small
radio projects. Read it, learn from it, let your AI tools chew on it — and then
build something of your own with it.

## Repositories
